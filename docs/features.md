# 功能清单

WebNest 围绕「以页面为文件夹」的可嵌套工作区组织笔记，按网站主机名归类。本文列出当前已实现的功能。

## 工作区树（页面树）

采用类 Notion 的「页面即文件夹」模型：

- **站点（site）** 是固定的顶层根节点，`id` 为 `site:<hostname>`，标题即主机名，不可重命名。
- **页面（page）** 可自由嵌套在站点或其他页面之下；每个节点都能保存 Markdown 内容并拥有子节点。
- 站点根**惰性创建**：仅当用户在当前站点首次新建页面（或手动「添加站点」）时才生成。

树的交互：

- 双击行内重命名（站点根不可改名）。
- 每行支持「在此节点下新建子页面」「删除页面」「展开 / 折叠」。
- **拖拽改父级**：把页面拖到另一节点下重新挂载，带后代环检测，避免把节点拖进自己的子树。移动时整棵子树的 `site` 会随新父级更新。
- **虚拟滚动**：行高 34px、上下各预渲染 8 行，基于 `scrollTop` + `ResizeObserver` 手动开窗；选中行会自动滚入可视区。

## Markdown 编辑器

双模式编辑（`MarkdownEditor.tsx`）：

- **源码模式**：CodeMirror 6，Markdown 语法高亮、行号、自动换行。
- **实时渲染 / 所见即所得模式**（默认）：TipTap / ProseMirror（`LiveMarkdownEditor.tsx`），基于 StarterKit + Link + Table（可调整大小）+ Placeholder + `@tiptap/markdown`。

内容统一以 **Markdown 字符串**存储与同步（兼容历史 ProseMirror JSON，见 [data-model.md](./data-model.md)）。

格式工具栏（`MarkdownToolbar.tsx`）：二级 / 三级标题、粗体、斜体、无序 / 有序列表、引用、代码块、链接、分割线，以及**表格大小选择器**（`TableGridPicker.tsx`）。源码模式走片段插入（`markdownInsert.ts`），实时模式走 TipTap 命令链。实时模式表格支持行 / 列缩放手柄。

**自动保存**：编辑后 1200ms 防抖写入，编辑器底部显示保存状态（保存中 / 已自动保存 / 失败）。

## 侧边栏与检索

- **搜索框**：按标题、站点、标签、来源、正文内容过滤页面树。
- **筛选**：全部 / 置顶 / 收藏 / 有标签。
- **排序**：循环切换 手动/更新 → 按创建 → 按标题；置顶项始终排在前面。
- **快捷区块**：可折叠的「收藏」「固定」区块，各自最多展示 4 项；显示与折叠状态持久化在 meta。
- **当前站点卡片**：展示活动标签页主机名，提供新建 / 定位操作。
- **底部栏**：添加站点、快捷区块设置、语言切换，以及数据操作（导出 JSON / 导出 Markdown / 导入 JSON）和实时存储占用读数。

## 收藏 / 固定 / 标签

- 每个页面有 `favorite`（收藏）与 `pinned`（置顶）布尔标记，`tags` 为字符串数组。
- 置顶项在各处排序中优先。
- 标签通过编辑器内的输入弹窗（`PromptDialog.tsx`）增删。

## 备份 / 导出 / 导入

- **JSON 备份**：格式标识 `opennote.workspace.v1`（`noteBackup.ts`），导出整棵工作区。
- **Markdown 导出**：将页面树展开为带元数据的多级标题文档。
- **JSON 导入**：整体替换当前工作区（带确认弹窗）。
- 下载文件名为 `opennote-workspace-<date>.json` / `.md`。

## 浏览器扩展行为

- 点击工具栏图标**打开侧边栏**（`openPanelOnActionClick`）。
- **右键「保存为笔记」**：选中网页文本后，菜单提供「新建笔记」与若干「追加到已有笔记」目标（按当前站点已有笔记动态生成，最多 20 项）。选区以文本 / HTML / Markdown 形式采集（注入 `capturePageSelection`）。
- 通过右键保存后，侧边栏会自动选中刚创建 / 更新的笔记（`chrome.storage.session` 中的 `pendingNoteSelect` + 运行时消息）。
- **跨面板状态同步**：监听 `chrome.storage.onChanged` 保持多处状态一致。
- **特殊页面**（`chrome://`、`chrome-extension://`、`about:`）不支持记录，会弹出提示。

## 国际化

应用 UI 支持简体中文（默认）与英文，可在侧边栏底部切换，并根据浏览器语言自动检测。详见 [i18n.md](./i18n.md)。
